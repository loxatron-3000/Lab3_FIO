## 🔧 Функциональность

Программа определяет тип треугольника по трем заданным сторонам:

1. **Ввод данных** - пользователь вводит длины трех сторон
2. **Проверка кеша** - поиск в БД (если есть, результат берется из БД)
3. **Вычисление** - если данных нет в БД, программа вычисляет тип треугольника
4. **Сохранение** - результат сохраняется в БД
5. **Отправка** - результат отправляется имитируемому email-серверу
6. **Вывод** - результат выводится пользователю

### Типы треугольников
- **Равносторонний** - все стороны равны
- **Равнобедренный** - две стороны равны
- **Разносторонний** - все стороны разные

### Обработка ошибок
- Стороны должны быть положительными числами
- Должно выполняться неравенство треугольника
